Miner Configuration

This is the basic guide on how to set up your miner for this pool.

Getting Started

To begin mining and use this pool, you'll need:

  • HTN Wallet address: Web Wallet
  • Crypto mining software that supports the HTN coin and Pyrinhash algorithm
  • Hardware to run it on. This can be your home PC, mining rig, ASIC miner, or cloud mining

HTN Wallet Address

A wallet address is required to receive payouts for your mined shares on this pool server. Once the total mined value surpasses the payout threshold, your coins will be sent to this wallet address.

Crypto Mining Software

To mine at this pool, you can use any miner supporting the etchash algorithm or HTN coin. Simply search for "HTN miner software" using a search engine, download the miner software, and configure your crypto miner accordingly.

Configuration Parameters:

  • POOL STRATUM ADDRESS AND PORT - stratum+tcp://fr.htn.pool.hoosat.fi:42494
  • YOUR_WALLET_ADDRESS - Your valid HTN wallet address
  • WORKERNAME - An optional workername to identify the miner or RIG
  • PASSWORD - Use 'x' or leave it blank
  • Optional: STATIC DIFFICULTY - To mine with a static (fixed) difficulty, use d=xxx as the password in your miner configuration, where xxx denotes your preferred difficulty.

For AMD Graphic Cards in a Windows System: Make sure you add this at the top of your .bat file:

setx GPU_FORCE_64BIT_PTR 0
setx GPU_MAX_HEAP_SIZE 100
setx GPU_USE_SYNC_OBJECTS 1
setx GPU_MAX_ALLOC_PERCENT 100
setx GPU_SINGLE_ALLOC_PERCENT 100

Configuration Examples

lolMiner Configuration
lolMiner.exe --algo PYRIN --pool stratum+tcp://fr.htn.pool.hoosat.fi:42494 --user YOUR_WALLET_ADDRESS.WORKERNAME --pass x
SRBMiner Configuration
SRBMiner-MULTI.exe --algorithm PYRIN --pool stratum+tcp://fr.htn.pool.hoosat.fi:42494 --wallet YOUR_WALLET_ADDRESS --worker WORKERNAME --password x
HiveOS Configuration

For HiveOS, use the following configuration:

  1. Click the flight sheet JSON below
  2. Go to Hive OS-farm (page that shows all workers)
  3. Click on Flight Sheets tab
  4. Click on Import from Clipboard
  5. Create Flight Sheet and add wallet
  6. Run flight sheet on workers!
